Machine Learning for Stock Markets

Knowledge Base



Prediction of assets behavior in financial series using machine learning algorithms



Autori

Diego Esteves da Silva, Julio Cesar Duarte

Anno

2013

In breve

Applicazione e comparazione di differenti metodi di machine learning sul mercato brasiliano. Si usa una finestra di dati (80 giorni) per predire l’andamento futuro (high o low)

Metodi

  • Tendency Keeper (non ML, usato per comparazione di base)
  • Naive Bayes
  • Logistic Regression
  • Support Vector Machine

Dataset

Indice BOVESPA, 107 record da Gennaio 2006 a Dicembre 2006. 80% training set, 20% test.

Attributi base: opening price (open), closing price (close), day maximum (max), day minimum (min) and volume (vol).

Risultati

L’algoritmo con le migliori performance è SVM. Tuttavia, tutti gli algoritmi hanno mostrato una capacità predittiva insoddisfacente (<70%)
