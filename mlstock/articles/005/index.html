Machine Learning for Stock Markets

Knowledge Base



SentAMaL- A Sentiment Analysis Machine Learning Stock Predictive Model



Autori

SA Bogle, WD Potter

Anno

2015

In breve

Esperimento di implementazione di un sistema ibrido di sentiment analysis e algoritmi di machine learning sul mercato azionario giamaicano.

Metodi

Sono stati provati i seguenti algoritmi per classificare i tweets in base al loro sentiment:

  • Naive Bayes Multinomial Text Classifier
  • Support Vector Machine
  • J48 Decision Tree

Naive Bayes Multinomial Text è stato alla fine scelto come migliore.

Dataset

Circa 2000 tweets tra Gennaio e Febbraio 2015, rappresentati un campione di esempi con polarità positiva, negativa o neutra riguardo la legalizzazione della marijuana per uso medicinale.

Risultati

  • 87% di accuracy per la predizione del trend
  • 0,99 di coefficiente di correlazione tra sentiment e andamento dei prezzi

Sentiment based models seem better suited for short term forecasts while machine learning approaches will likely outperform them on long term forecasts.
